# Dziennik prac

Tutaj zapisuje się postęp projektu, checkpointy i istotne decyzje.

## Cel
- śledzenie zmian w czasie,
- dokumentowanie odkryć i eksperymentów,
- tworzenie historii rozwoju projektu.

## Format wpisów
Każdy wpis powinien zawierać datę, numer checkpointu, krótki opis wykonanych prac i najważniejsze wnioski.
