# ARCANE

## Misja
ARCANE to lokalna platforma orkiestracji agentów AI, zaprojektowana do pracy w środowisku Purple Team Operations. Projekt ma dwa równorzędne cele:
- zbudować profesjonalną platformę AI,
- rozwijać autora projektu jako architekta systemów AI i specjalisty cyberbezpieczeństwa.

## Główne założenia
- Local First — rozwój i działanie lokalnie, bez zależności od zewnętrznego serwera.
- AI Orchestrator — centralny mechanizm zarządzania wieloma agentami.
- Multi-Agent System — współpraca agentów w ramach wspólnego celu.
- Purple Team — wsparcie dla ofensywnych i defensywnych scenariuszy operacyjnych.
- Knowledge Driven — decyzje oparte na wiedzy, kontekście i historii.
- Memory First — pamięć robocza, semantyczna, epizodyczna i decyzyjna jako fundament działania.
- Documentation First — każda decyzja i moduł mają swoje miejsce w dokumentacji.
- Architecture Before Code — najpierw projekt, potem implementacja.
- Learning Through Building — projekt jest jednocześnie platformą i procesem nauki.

## Wizja projektu
ARCANE nie będzie klasycznym dashboardem. Będzie miastem.
Każda dzielnica odpowiada za inny obszar odpowiedzialności, a pod miastem biegnie metro komunikacji między modułami.

### Dzielnice miasta
- City Hall — Chief Orchestrator
- Cognitive District — reasoning, planning, reflection
- Memory District — różne typy pamięci
- Knowledge District — knowledge graph, RAG, research
- Engineering District — developer agents
- Security District — Purple Team
- Documentation District — dokumentacja i wiedza o projekcie
- Academy — nauka i rozwój kompetencji
- Architecture District — projektowanie architektury systemu
- Testing District — testy i walidacja

## Zasada pracy
Nie kopiujemy projektów ani kodu. Analizujemy, rozumiemy, projektujemy i budujemy własne rozwiązanie inspirowane nowymi wzorcami architektonicznymi.
