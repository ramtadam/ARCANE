# Architecture

Ten katalog gromadzi dokumentację architektury wysokopoziomowej projektu ARCANE.

## Główne obszary
- City Hall — orchestracja i koordynacja całego systemu,
- Cognitive District — planowanie, rozumowanie i refleksja,
- Memory District — różne poziomy pamięci,
- Knowledge District — wiedza, graph i retrieval,
- Communication Layer — metro komunikacyjne między agentami,
- Security District — mechanizmy Purple Team,
- Observability — monitoring, logi i wizualizacja.

## Podejście
Architektura jest projektowana jako organizm miejskiego typu, a nie jako zbiór przypadkowych modułów. Każdy komponent ma swoją rolę, granice odpowiedzialności i sposób komunikacji z resztą systemu.
