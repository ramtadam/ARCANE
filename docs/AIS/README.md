# AIS

Agent Interaction Specifications opisują, jak agenci komunikują się między sobą oraz z otoczeniem.

## Główne obszary
- protokoły interakcji,
- przepływy komunikatów,
- reguły koordynacji,
- obsługa błędów i retry,
- logowanie komunikacji i monitorowanie kolejki wiadomości.

## Koncepcja komunikacji
Pod miastem biegnie metro komunikacji. To warstwa, która umożliwia przepływ informacji między dzielnicami bez bezpośredniego „przeskakiwania” między modułami.
