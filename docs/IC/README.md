# Implementation Candidates

Tutaj zapisuje się konkretne pomysły i propozycje wdrożeń, które mogą zostać później rozwinięte do pełnej implementacji.

## Cele
- gromadzenie pomysłów na rozwiązania,
- porządkowanie koncepcji bez natychmiastowej decyzji o wdrożeniu,
- łatwiejsze przechodzenie od idei do realizacji,
- zachowanie przejrzystości między etapem analizy a etapem implementacji.

## Przepływ decyzji
Pomysł -> TAR -> AIS -> ADR -> Implementation Candidate -> implementacja.
