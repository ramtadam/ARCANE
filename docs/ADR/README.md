# ADR

Katalog Architectural Decision Records zawiera decyzje architektoniczne projektu ARCANE.

## Zasada
Każda istotna decyzja powinna być opisana wraz z kontekstem, problemem, alternatywami i uzasadnieniem wyboru. Dokumenty ADR mają wspierać zrozumienie, dlaczego system jest zbudowany w dany sposób.

## Przykładowe obszary decyzji
- wybór modelu orkiestracji,
- projekt komunikacji między agentami,
- organizacja pamięci,
- struktura modułów i warstw,
- podejście do testów i dokumentacji.
